## Setup

```bash
git clone https://github.com/Junaid2005/encode-hack-nov.git
cd encode-hack-nov

python3 -m venv venv

# Linux / macOS
source venv/bin/activate

# Windows
venv\Scripts\activate

pip install -r requirements.txt

streamlit run src/frontend/app.py
```
